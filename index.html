iDEX ADITI 4.0 — CEW Platform
Challenge: Cognitive Electronic Warfare (CEW) Platform — ADITI 4.0 PS-5, Indian Army. Autonomous EW System for multi-sensor data fusion, predictive analysis, and ECM execution. Proposed duration: 24 months.
CEW System — 4-Layer Architecture
Layer 1
Physical Layer (Full Spectrum Detection)
Modular GaN SDR daughterboards — 50 MHz to 40 GHz (VHF to Ka Band). VHF/UHF drones, L/S/C air defense radars, X/Ku/Ka fire-control + Satcom. Real-time IQ streaming to FPGA core.
Layer 2
Cognitive Processing (TinyML GNN)
RF + acoustic + terrain data fusion. GNN intent prediction <1 ms latency ("Lock-on" vs "Search"). Adversarial defense via secondary detector GNN for spoofed signals. Zero-OS FPGA — deterministic latency.
Layer 3
Security & Decoy (TAM + I2P)
TAM monitors power + EM emissions → injects dummy traffic + jitter + key rotation on side-channel detection. I2P Decoy Engine triggers 5–10 km NLOS fake heartbeat nodes, causing adversary DF systems to triangulate false locations.
Layer 4
Action Layer (Jx Module)
Adaptive ECM execution: noise jamming, range-gate pull-off deception, frequency denial, or intelligent silence — based on GNN intent prediction.
Key Differentiators
I2P Decoy Network
First-of-kind use of I2P routing to distribute fake signal generation across 5–10 km NLOS mesh — true C2 location invisible. 2 pending patents.
Active TAM Security
Moves beyond passive encryption to actively neutralize side-channel and traffic analysis attacks in real-time via constant-power noise + dummy traffic.
Zero-OS Architecture
Eliminates software attack surfaces — pure FPGA-native ensuring deterministic <1ms latency. No OS = no OS-level attack vector.
Temporal Steganography
Covert data embedding in CPPM timing patterns — a layer of concealment beyond encryption and RF evasion.
